# Inspiration

Reference architectures to study once your brain has real content. Borrow the ideas that fit how
*you* work; don't copy wholesale.

- **second-brain-template** — https://github.com/Loringtonian/second-brain-template — a fully
  built-out version of this architecture: a 7-rule context router, A/B/C document templates, ~30
  skills, deterministic hooks, privacy-depth gating. This seed is the lightweight front door; that
  template is where it can grow.
